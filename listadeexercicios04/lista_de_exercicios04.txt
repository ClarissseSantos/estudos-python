1. Sorteie 10 inteiros entre 1 e 100 para uma lista e descubra o maior e o menor valor, sem usar as funções max e min. 

import random
lista = []
maior = 0
menor = 100

for i in range(10):
	n = random.randint(1,100)
	lista.append(n)

	if n < menor: 
		menor = n
	if n > maior:
		maior = n

lista.sort()

2. Sorteie 20 inteiros entre 1 e 100 numa lista. Armazene os números pares na lista PAR e os números ímpares na lista IMPAR. Imprima as três listas. 

import random

lista = []
par = []
impar = []

for i in range(20):
	n = random.randint(1,100)
	lista.append(n)

	if n % 2 == 0: 
		par.append(n)
	else:
		impar.append(n)

lista.sort()
par.sort()
impar.sort()

print ("\nlista = ", lista)
print ("par = ", par)
print ("impar = ", impar, "\n")
